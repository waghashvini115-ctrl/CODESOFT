## Description
This project analyzes historical movie data and uses machine learning techniques
to predict movie ratings based on different features.

## Dataset
The dataset contains movie-related information such as ratings and other attributes.

## Tools & Technologies
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib

## Machine Learning Model
Regression model is used to predict movie ratings.
